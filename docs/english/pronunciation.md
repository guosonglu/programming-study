# 规律

- `元音音标+辅音音标`发一个音
- 一个元音音标对应一个音节，一个单词的最后一个音节为 `重读音节`,即发四声
- `元音+r`发长元音。特殊情况有war和wor，它们与ar和or发音是不一样的
