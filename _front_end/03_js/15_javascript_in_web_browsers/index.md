# Web编程基础

{: .note-title}
> 客户端JavaScript和服务端JavaScript
> 
> - `客户端JavaScript`就是在`浏览器`中运行的JavaScript代码
> - `服务端JavaScript`就是运行在服务器上的程序。

`Node`有自己唯一的实现，也有自己唯一的官方文档。相对而言，`Web API`则是通过`主要浏览器厂商的共识来定义的`。
