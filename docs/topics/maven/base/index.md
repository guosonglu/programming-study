# Maven基本教程

## 简介

Apache Maven，是一个软件（特别是Java软件）`项目管理`及`自动构建`工具，由Apache软件基金会所提供。Maven也可被用于构建和管理各种项目，例如C#，Ruby，Scala和其他语言编写的项目。Maven曾是Jakarta项目的子项目，现为由Apache软件基金会主持的独立Apache项目。

Maven解决了软件构建的两方面问题：

- 一是软件是如何构建的
- 二是软件的依赖关系
