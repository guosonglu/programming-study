# latex公式对照表

| 运算名称         | 效果             | 公式             |
|--------------|----------------|----------------|
| 加            | $a+b$          | `a+b`          |
| 减            | $a-b$          | `a-b`          |
| 乘            | $a \times b$   | `a \times b`   |
| 除(division)  | $a \div b$     | `a \div b`     |
| 分数(fraction) | $\frac {a}{b}$ | `\frac {a}{b}` |
| 幂            | $a^b$          | `a^b`          |
| 根号(square)   | $\sqrt a$      | `\sqrt a`      |
|              | $\sqrt[b]a$    | `\sqrt[b]a`    |
| 不等于          | $a \neq b$     | `a \neq b`     |
| 空格           | $a \quad b$    | `a \quad b`    |
| 圆周率          | $\pi$          | `\pi`          |
