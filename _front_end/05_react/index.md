# 脚手架创建项目

```bash
# 使用npx创建项目
npx create-react-app my-app

# 切换到项目目录
cd my-app

# 启动项目
npm start
```

# JSX

{: .note-title}
> JSX
> 
> JSX是一种JavaScript的语法扩展，它的语法看起来像是在JavaScript中嵌入了HTML代码，但实际上它是在JavaScript中创建了一个虚拟DOM对象。
