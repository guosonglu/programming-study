# latex公式对照表

| 运算名称 | 效果             | 公式             |
|------|----------------|----------------|
| 加    | $a+b$          | `a+b`          |
| 减    | $a-b$          | `a-b`          |
| 乘    | $a \times b$   | `a \times b`   |
| 除    | $a \div b$     | `a \div b`     |
| 分数   | $\frac {a}{b}$ | `\frac {a}{b}` |
| 幂    | $a^b$          | `a^b`          |
| 根号   | $\sqrt a$      | `\sqrt a`      |
|      | $\sqrt[b]a$    | `\sqrt[b]a`    |
